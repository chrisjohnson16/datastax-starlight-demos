Running Rabbit

docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.10-management



Direct with Rabbit

java -jar target/s4r-demo-0.0.1-SNAPSHOT.jar rabbit receive

java -jar target/s4r-demo-0.0.1-SNAPSHOT.jar rabbit send
